## 4. 世界観

### 4.1 舞台

| 項目 | 内容 |
| --- | --- |
| **場所** | 廃墟になった洋館 |
| **時間帯** | 夜（月明かりあり） |
| **電気** | 一部通っている箇所もあるが、基本的に暗い |
| **雰囲気** | 薄暗く静かな廃墟 |

### 4.2 登場キャラクター

| キャラ | 設定 |
| --- | --- |
| **影猫** | 光の猫と「対の存在」。影に潜み、物に変身する能力を持つ |
| **光の猫** | 影猫と「対の存在」。影の中を通れない |
| **泥棒** | 光の猫を売るために攫った。廃墟で安心してひと眠り中 |
| **影ネズミ** | 廃墟に自然発生した影の怪物。光を憎み、消そうとする |

### 4.3 ストーリー

泥棒が光の猫を攫い、廃墟の屋敷に逃げ込んだ。売り飛ばすつもりだが、今は安心してひと眠り。影猫は対の存在である光の猫を救うため、屋敷に侵入。廃墟には影ネズミが巣食っており、光を消そうと襲ってくる。影猫は自分が苦手な光を守りながら、光の猫を救出する。

## 5. プレイヤーキャラクター：影猫

### 5.1 概要

| 項目 | 内容 |
| --- | --- |
| **立ち位置** | 光の猫と「対の存在」。自ら直接物を動かす力はない |
| **能力** | 影に入り込み、そのオブジェクトに変身して操る |
| **ビジュアル** | 普段は黒猫。変身後はオブジェクトに猫耳が生えた見た目 |

### 5.2 操作仕様

| アクション | キーボード | コントローラー | 詳細 |
| --- | --- | --- | --- |
| **移動** | WASD | Lスティック | 変身前後どちらも同じ |
| **変身/解除** | Shift | ZL | 影の上で押すと変身、再押しで解除 |
| **転倒** | Space | ZR＋Rスティック | 共通アクション。ネズミ撃破やギミック用 |
| **固有アクション** | Ctrl | L or R | オブジェクト固有の動作（音を鳴らす等） |
| **リセット** | R長押し | ＋ボタン長押し | ステージ全体を初期化 |
| **ポーズ** | Esc | −ボタン長押し | ポーズメニューを開く |
| **決定/進む** | Enter | Aボタン | メニュー操作、テキスト送り |

### 5.3 変身システム

| 項目 | 仕様 |
| --- | --- |
| **変身条件** | オブジェクトの影テクスチャの上でShift（ZL） |
| **変身可能表示** | 影が点滅して変身可能であることを示す |
| **複数オブジェクト時** | 自動処理、点滅で選択 |
| **変身解除** | 同じボタン再押しで即座に元に戻る |
| **変身中の移動** | 通常移動と同じ操作・同じ速度 |
| **変身可能時間** | 20秒 |
| **時間表示** | 時計UI（針が回って残り時間を示す） |
| **時間切れ処理** | 強制解除＋クールダウン（5秒間変身不可） |
| **途中解除時** | クールダウンなし、すぐ再変身可能 |

### 5.4 光の扱い

| 項目 | 仕様 |
| --- | --- |
| **光の中の移動** | **通行不可**（壁扱い） |
| **HP/ダメージ** | **なし**（管理不要） |

### 5.5 リセット機能

| 項目 | 仕様 |
| --- | --- |
| **操作** | R長押し / ＋ボタン長押し |
| **効果** | ステージ全体を初期状態に戻す |
| **戻るもの** | 家具の位置、ネズミ、穴、発電機の耐久値、光の猫の位置（全部） |

### 5.6 カメラ仕様

| 項目 | 仕様 |
| --- | --- |
| **視点** | 三人称・固定アングル |
| **回転** | 不可（Rスティック操作なし） |
| **表示範囲** | プレイヤーの周囲のみ見える形 |
| **変身中** | 同じカメラ仕様 |

## 9. 敵キャラクター：影ネズミ

### 9.1 概要

| 項目 | 内容 |
| --- | --- |
| **設定** | 廃墟に自然発生した影の怪物 |
| **目的** | 光を憎み、消そうとする |
| **役割** | プレイヤーの行動を妨害し、緊張感を与える |
| **ステージ上の最大数** | 3匹（これ以上は出現しない） |

### 9.2 AIアルゴリズム

| 項目 | 仕様 |
| --- | --- |
| **経路探索** | グリッドベース |
| **経路再計算** | 光が出現したら即座に再計算 |
| **音への反応** | 鳴った瞬間に向かう、攻撃中でも即座に中断 |
| **音源での行動** | 音源の周りをうろつく |
| **音停止後** | 2秒ぼーっとしてから発電機に向かう |
| **複数音源時** | 先に鳴った方に向かう |
| **衝突判定** | あり（ネズミ同士重ならない） |
| **発電機への群がり** | 横に並ぶ（最大3匹） |

### 9.3 行動ロジック（優先順位）

1. **音（高優先）**: ラジカセ等の音が鳴ると、攻撃を中断して音源に集まり、周りをうろつく
2. **破壊（基本）**: 音がない場合、「発電機」に向かって最短ルートで移動し、攻撃する
3. **回避**: 通れない場所は迂回する

### 9.4 認識範囲

| 項目 | 仕様 |
| --- | --- |
| **視界・認識範囲** | 5m |
| **音への反応範囲** | 5m |
| **発電機の認識** | 視界内で認識 |

### 9.5 移動制限

| 通れない場所 | 備考 |
| --- | --- |
| **強い光（ライトの照射範囲）** | 光の壁として機能 |
| **光の道** | 光の猫の通り道には侵入不可 |
| **坂** | 地形による制限 |
| **落とし穴** | 地形による制限 |

### 9.6 撃退方法

| 方法 | 効果 | 備考 |
| --- | --- | --- |
| **家具の下敷き（転倒）** | **一撃死** | 命中で即撃退 |
| **大型オブジェクトで空振り** | **スタン（2〜3秒）** | 振動による範囲攻撃 |

### 9.7 撃退演出

| 演出 | 内容 |
| --- | --- |
| **撃退時** | その場で息絶える |
| **エフェクト** | 天使の輪っかが頭につき、天に召されていく |

### 9.8 出現ルール

| 項目 | 仕様 |
| --- | --- |
| **出現条件** | 攻略に不要なオブジェクトを動かすと、その下の穴から出現 |
| **出現数** | 1回のミスで1匹 |
| **穴の状態** | 一度開いたら開いたまま |
| **追加出現** | 時間経過で穴から追加で湧いてくる（5秒間隔） |
| **総数上限** | ステージ上に最大3匹まで |

| Stage | 初期ネズミ | 用途 |
| --- | --- | --- |
| 1 | 0匹 | なし（チュートリアル、罠オブジェクトもなし） |
| 2 | 0匹 | なし |
| 3 | 3匹 | スイッチの重し用 |

## 10. 救出対象：光の猫

| 項目 | 仕様 |
| --- | --- |
| **立ち位置** | 影猫と「対の存在」。泥棒に攫われ、檻に閉じ込められている |
| **ビジュアル** | 白猫（檻の中で座っている） |
| **移動制限** | 影の中を通れない（影猫と逆） |
| **AI** | イベント以外は移動しない |
| **移動トリガー** | 光の道（檻→出口）が完成したら自動で歩き出す |
| **光の道が途切れた場合** | 立ち止まる（引き返さない） |
| **ネズミとの接触** | GAMEOVER |
| **脱出距離** | ステージごとに異なる |
| **脱出演出** | 奥に進んでいく → クリア |

## 11. ステージ構成とギミック（全4ステージ）

### Stage 1: 物理と影（Tutorial）

| 項目 | 内容 |
| --- | --- |
| **目的** | 基本操作と「家具を倒す」仕組みを学ぶ |
| **攻略概要** | 本棚に変身して転倒 → 倒れた本棚が橋になり壁を越える |

### Stage 2: 伸びる影（Light Puzzle）

| 項目 | 内容 |
| --- | --- |
| **目的** | 光源を動かして「影の道」を作る。カーテンの使い方を学ぶ |
| **攻略概要** | カーテンで光を遮断 → ライトで柱に光を当てる → 柱の影が道になる |

### Stage 3: 音の誘導（Stealth & Lure）

| 項目 | 内容 |
| --- | --- |
| **目的** | 敵を「音」でコントロールする |
| **攻略概要** | ラジカセで音を鳴らす → ネズミを誘導してスイッチに乗せる → 扉が開く |

### Stage 4: 光の道（Final）

| 項目 | 内容 |
| --- | --- |
| **目的** | 光の猫が安全に脱出できる「光の道」を作る |
| **攻略概要** | ネズミをスイッチに誘導して檻を開ける → ライトとカーテンで光の道を作る → 光の猫が自動で脱出 |
